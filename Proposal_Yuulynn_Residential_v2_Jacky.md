## 二、痛點頁：頂級地段的 4 個焦慮

### 焦慮 1：空間被壓縮
大安區精華地段，基地面積通常不大，戶型配置與公設比例需極致優化。
**數據**：基地面積縮減 15%，但買方對公設品質要求提升 40%。

### 焦慮 2：法規審查延長
容積率、建蔽率、綠覆率、無障礙、防火避難……每次退件延長 2–3 週。
**數據**：台北市建筑審查平均退件 2.3 次，每次修改成本約 15–30 萬。

### 焦慮 3：競品同質化
同區域 3 個豪宅案同時開賣，買方為何選擇玉霖？
**數據**：和平東路半徑 500m 內，同期競品 ≥ 3 個。

### 焦慮 4：買方期待變革
67% 的豪宅買方在詢問時會問「這是智慧建築嗎？」
**數據**：智慧建築認證建案溢價 5–8%。

> **結論**：頂級地段不缺買家，缺的是「賣得出差異化」的建案。

## 四、FM 專利主軸：賣房子時，20 年物管已經準備好

### 為什麼是「專利」？

協勤資訊的 BIM FM 系統於 **2012 年開發**，已在 **台積電**、**慈濟醫療**、**新北市政府** 等重量級客戶上線運作，並獲得：
- 土木水利 BIM 技術獎
- 第九屆內政部智慧綠建築競賽獎項

### FM 資料層：從設計階段就植入

| 階段 | 產出物 | 對應 FM 資料層 |
|:---|:---|:---|
| 規劃 | 空間配置方案 | 空間動態屬性資料庫 |
| 設計 | 設備點位、管線佈置 | 設備完整屬性、BIM ID 對應 |
| 施工 | 竣工模型、設備清冊 | 完整資產盤點資料 |
| 移交 | COBie 資料轉換 | 營運移交即插即用 |
| 營運 | IoT 感測器、BAS 點位 | 探識空間 FDD 主動式預警 |

### 三層架構：IoT + 大數據 + BIM

```
第一層：資料取得（IoT/BAS/手機 APP）
第二層：資料分析（FDD 故障診斷、統計管制、AI 預測）
第三層：資料呈現（3D 儀表板、AR360 可視化、雲端管理）
```

## 五、地段分析：和平東路 × 復興北路

### 地理位置
- **行政區**：台北市大安區
- **交叉口**：和平東路一段 × 復興北路
- **周邊地標**：
  - 大安森林公園（約 800m）
  - 台北科技大學、師大附中
  - 捷運科技大樓站、大安站、忠孝復興站
  - 遠東 SOGO、明曜百貨商圈

### 市場定位
- **區域屬性**：台北市頂級豪宅聚落，文教區與商業區交匯
- **房價行情**：
  - 大安 ONE（新建案）：均價約 161 萬/坪
  - 和平大樓：均價約 122 萬/坪
  - 周邊中古屋：100–140 萬/坪
- **目標客群**：高資產家庭、科技新貴、重視教育與生活品質的換屋族

### 開發挑戶与機會
1. **基地條件受限**：市區精華地段，基地面積通常不大，需極致空間規劃
2. **法規嚴格**：容積率、建蔽率、綠覆率、無障礙規定要求高
3. **鄰近施工干擾**：繁忙路口，施工動線與噪音管制嚴苛
4. **買方期待高**：頂級地段買方對智慧化、永續性、公設品質要求極高
5. **銷售競爭激烈**：同區域多個豪宅案競爭，差異化是關鍵

> **機會**：這裡的買方願意為「智慧、永續、全齡」多付 5–8% 的溢價。
