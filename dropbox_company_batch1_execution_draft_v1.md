## 建議的 Dropbox 搬遷指令概念

> 先示意，不直接執行。

### 建資料夾
```json
POST /dropbox/2/files/create_folder_v2
{ "path": "/公司資料/01_營運管理", "autorename": false }
```

### 移動資料夾
```json
POST /dropbox/2/files/move_v2
{
  "from_path": "/公司資料/報價單",
  "to_path": "/公司資料/02_業務銷售/報價單"
}
```

## 執行前最後確認事項

在真正搬之前，應再確認：
1. 目標路徑不存在同名資料夾，避免覆蓋或混併
2. `專案相關資料`、`協勤部門管理資料` 這種大資料夾搬遷可能需要較長時間
3. 若有外部同步流程或捷徑引用原路徑，搬遷後可能需要調整
4. 建議先搬 3~5 個低風險資料夾做試跑
