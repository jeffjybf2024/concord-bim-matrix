## 三、地段分析：和平東路 × 復興北路

### 地理位置
- **行政區**：台北市大安區
- **交叉口**：和平東路一段 × 復興北路
- **周邊地標**：大安森林公園（約 800m）、台北科技大學、師大附中、捷運科技大樓站、遠東 SOGO

### 市場定位
- **區域屬性**：台北市頂級豪宅聚落，文教區與商業區交匯
- **房價行情**：大安 ONE 均價約 161 萬/坪、周邊中古屋 100–140 萬/坪
- **目標客群**：高資產家庭、科技新貴、重視教育與生活品質的換屋族

### 開發挑戰
1. **基地條件受限**：市區精華地段，基地面積通常不大
2. **法規嚴格**：容積率、建蔽率、綠覆率、無障礙規定要求高
3. **鄰近施工干擾**：繁忙路口，施工動線與噪音管制嚴苛
4. **買方期待高**：頂級地段買方對智慧化、永續性、公設品質要求極高
5. **銷售競爭激烈**：同區域多個豪宅案競爭，差異化是關鍵

> **機會**：這裡的買方願意為「智慧、永續、全齡」多付 5–8% 的溢價。

## 九、設計階段（3 項展開）

## 十一、業界創舉一：4D + 現況比對 消費者視覺電子合約

### 概念
傳統預售合約只有平面圖和文字說明，買方無法即時掌握施工進度。

**我們做的**：將 BIM 4D 施工模擬與現場實景照片結合，買方透過專屬 App：
- 隨時查看各樓層施工進度（3D 模型 + 時間軸）
- 比對設計模型與現場現況照片
- 繳款進度與工程里程碑自動掛鉤

### 技術實現架構

```
BIM 4D 模型（Revit/Navisworks）
    ↓
進度資料 API 串接（每周自動更新）
    ↓
現場照片 AI 比對（ReCap + 電腦視覺）
    ↓
消費者 App（Web/App，單一登入）
    ↓
繳款里程碑自動觸發（與 ERP 整合）
```

### 買方體驗流程

```
開工 → 基礎完成 → 結構體完成 → 外牆完成 → 機電完成 → 裝修完成 → 交屋

App推播  App推播     App推播      App推播    App推播    App推播   AR驗收
+現況照  +現況照      +現況照       +現況照     +現況照     +現況照    模型比對現場
```

## 十二、業界創舉二：AR/MR 竣工驗收

### 概念
傳統竣工驗收由專業人員執行，買方只能看紙本報告，無法親自確認細節。

**我們做的**：買方戴上 AR 眼鏡（或用手機 AR），將 BIM 竣工模型疊合現場實景：
- 牆面平整度：模型與現場比對，偏差自動標註
- 管線位置：隱藏管線在 AR 中顯示，確認與設計一致
- 設備規格：掃描設備 QR Code，即時顯示 BIM 屬性資料
